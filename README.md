# 제3회 전기안전공모전

## Frontend
### 실행 방법
```
# pnpm 설치 (최초 1회)
npm install -g pnpm

# 프로젝트 디렉토리 이동
cd frontend

# 패키지 설치
pnpm install

# 개발 서버 실행
pnpm dev

# 프로젝트 빌드
pnpm build
```

### 기술 스택
React (with Vite) – 빠른 프론트엔드 빌드 도구
JavaScript – 주요 개발 언어
Antd – UI 구성요소 라이브러리
Zustand – 전역 상태 관리 라이브러리
PNPM – 빠르고 효율적인 패키지 매니저
